# 1. Project Plan & Timeline

## Project Overview
The goal was to build a full-stack "AI-Powered Personal Study Assistant" over the course of 8 weeks. The project followed an Agile methodology, broken down into distinct phases.

## Timeline & Milestones

### Phase 1: MVP & Foundation (Weeks 1-2)
*   **Goal**: Establish core architecture and document management.
*   **Deliverables**:
    *   [x] FastAPI Backend Setup & Docker Configuration.
    *   [x] React Frontend Scaffolding.
    *   [x] Document Upload API (PDF/PPTX parsing).
    *   [x] Basic "Library" View.

### Phase 2: AI Study Tools (Weeks 3-4)
*   **Goal**: Implement the core AI value proposition.
*   **Deliverables**:
    *   [x] AI Service Integration (Anthropic/Ollama).
    *   [x] Flashcard Generation Endpoint & UI.
    *   [x] Quiz Generation Endpoint & UI.

### Phase 3: Interactive Features (Weeks 5-6)
*   **Goal**: Enhance user engagement and organization.
*   **Deliverables**:
    *   [x] Conversational AI (Chat with Documents).
    *   [x] Document Search & Filtering.
    *   [x] Auto-Categorization of uploads.

### Phase 4: Progress Tracking (Week 7)
*   **Goal**: Add analytics and motivation.
*   **Deliverables**:
    *   [x] `StudySession` Data Model.
    *   [x] Analytics API (Streaks, Duration).
    *   [x] Dashboard Component with Charts (Recharts).

### Phase 5: Refinement & Spaced Repetition (Week 7)
*   **Goal**: Polish and advanced learning algorithms.
*   **Deliverables**:
    *   [x] Spaced Repetition (SM-2) Algorithm.
    *   [x] Flashcard "Review Mode" (Rating System).
    *   [x] Final UI Polish & Bug Fixes.

### Phase 6: Deployment (Week 7)
*   **Goal**: Production readiness and cloud hosting.
*   **Planned Deliverables**:
    *   [ ] Docker Containerization (Frontend & Backend).
    *   [ ] Cloud Deployment (AWS/GCP).
    *   [ ] SSL/TLS Security & Domain Setup.

### Phase 7: Usability Testing (Week 8)
*   **Goal**: Evaluate user experience and identify improvements.
*   **Deliverables**:
    *   [ ] Usability Testing Protocol (Survey & Tasks).
    *   [ ] Feedback Collection .
    *   [ ] Bug Fixes & Refinements based on feedback.

### Phase 8: Documentation (Week 8)
*   **Goal**: Comprehensive project handover.
*   **Deliverables**:
    *   [ ] Functional Web Application (Source Code).
    *   [ ] API Documentation (Swagger UI).
    *   [ ] User Manual / README.
    *   [ ] Final Project Report.
    *   [ ] Presentation Slide Deck.

### Phase 9: Maintenance & Updates (Future)
*   **Goal**: Long-term sustainability.
*   **Planned Deliverables**:
    *   [ ] User Feedback Loop Integration.
    *   [ ] Performance Monitoring (New Relic/Datadog).
    *   [ ] Regular Security Audits.
